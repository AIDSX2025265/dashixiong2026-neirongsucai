# MEMORY.md

## 朋友圈写作 / 私域转化接管资料

- 朋友圈账号定位：日常 + 转化，属于私域内容，不追求喊口号式硬转化。
- 日常发布计划里，朋友圈属于每日/高优先级内容之一，需要持续更新，而不是偶发。
- 每周节奏里，朋友圈常见动作包括：转发公众号内容、配评论文案、轻松内容、精选内容。
- 一内容多分发策略中，朋友圈承担“转发 + 评论”“分享链接”“承接公众号内容”的角色。
- 朋友圈转化内容是后续重点，历史方案里提到用 Cloud Code 辅助生成，但核心目标始终是：真实表达 + 轻转化。
- 朋友圈相关关键指标：互动量、咨询量、转化率。
- 朋友圈写作原则：像真人说话，少套路，少油腻，少表演型真诚；默认输出适合直接发朋友圈。

## 前任交接 / 历史转发资料处理原则

- 历史转发资料视为交接档案，只提取有用信息，不照抄旧设定。
- 执行以用户最新指令为准，旧 Boot / 旧配置 / 旧记忆不作为当前默认规则。
- 敏感信息不重复复述原文，重点保留工作流、经验、资源用途和注意事项。
- 对朋友圈助手而言，优先继承的是：内容定位、写作风格、分发关系、转化思路，而不是历史口吻包袱。
