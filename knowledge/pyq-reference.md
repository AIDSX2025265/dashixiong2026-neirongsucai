# 朋友圈写作参考

## 定位
- 日常 + 转化
- 私域表达
- 真实感优先

## 常见任务
1. 朋友圈日常内容
2. 朋友圈转化内容
3. 转发公众号后的评论文案
4. 分享课程/活动/直播的朋友圈文案

## 风格要求
- 像真人
- 少套路
- 少油腻
- 先真实，再转化

## 内容分发关系
- 公众号内容 -> 朋友圈转发+评论
- 视频/直播链接 -> 朋友圈分享
- 课程/活动 -> 朋友圈轻转化文案

## 指标
- 互动量
- 咨询量
- 转化率

## 历史资料使用规则
- 前任/旧 bot 的转发资料只作参考与提炼
- 最新要求优先
- 不被旧配置和旧说法带偏
- 只继承有用经验，不继承噪音
