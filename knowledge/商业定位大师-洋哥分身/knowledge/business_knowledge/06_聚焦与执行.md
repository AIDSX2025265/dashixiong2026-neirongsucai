# 聚焦与拒绝内耗

> 基于洋哥行动家密训内容

---

## 核心认知：内耗的来源

### 什么是内耗

**内耗 = 想得多 + 做得少 + 选择困难**

内耗的本质是：你的精力和时间消耗在了内部纠结上，而不是外部行动上。

### 内耗的三个主要来源

**来源一：选择过多**
- 什么都想做
- 什么都想要
- 结果什么都做不好

**来源二：定位不清**
- 不知道自己要什么
- 不知道自己的目标
- 在多个选项中摇摆

**来源三：执行力不足**
- 想得多，做得少
- 完美主义拖延
- 害怕失败不敢开始

## 洋哥的聚焦方法

### 聚焦的本质

**聚焦 = 选择不做什么比选择做什么更重要。**

聚焦不是说只做一件事，而是说：把有限的精力聚焦在最重要的事情上。

### 聚焦的四个步骤

**步骤一：明确核心目标**

**问题**：你真正想要的是什么？

**操作**：
1. 列出你想要的所有东西
2. 按重要性排序
3. 找到最核心的1-3个目标
4. 其他都暂时放下

**关键**：不是所有的目标都值得追求，只追求最重要的。

**步骤二：做减法**

**问题**：你能砍掉哪些不重要的？

**操作**：
1. 列出你现在正在做的所有事情
2. 问自己：这件事情对我的核心目标重要吗？
3. 砍掉不重要的事情
4. 把精力集中在重要的事情上

**关键**：砍掉不重要的，才能把重要的做好。

**步骤三：单点突破**

**问题**：在哪个点做到极致？

**操作**：
1. 选择一个细分领域
2. 选择一个核心能力
3. 选择一个目标用户
4. 在这个点上做到极致

**关键**：不要追求全面，追求单点突破。

**步骤四：持续行动**

**问题**：如何用行动消除焦虑？

**操作**：
1. 不追求完美，先完成
2. 快速试错，快速迭代
3. 用行动验证假设
4. 在行动中调整方向

**关键**：行动是消除焦虑的最好方法。

## 如何拒绝内耗

### 方法一：二分法决策

**核心思想**：对于不重要的事情，快速决策

**操作**：
1. 判断：这件事情重要吗？
2. 如果不重要，快速做决策（比如抛硬币）
3. 如果重要，收集信息后做决策
4. 不要过度纠结

**关键**：把纠结的时间用来行动。

### 方法二：时间盒限制

**核心思想**：给决策和执行设定时间限制

**操作**：
1. 给决策设定时间限制（比如10分钟）
2. 给任务设定时间限制（比如1小时）
3. 时间到了就停止，进入下一个
4. 在限制内追求效率

**关键**：时间限制能消除完美主义。

### 方法三：最小行动启动

**核心思想**：用最小的行动启动自己

**操作**：
1. 把大任务拆解成小任务
2. 找到最小的那个行动
3. 立即执行这个最小行动
4. 用小行动带动大行动

**关键**：最小行动能克服启动阻力。

### 方法四：接受不完美

**核心思想**：接受不完美，在行动中完善

**操作**：
1. 承认第一次不可能完美
2. 接受"先完成"的理念
3. 在行动中收集反馈
4. 在迭代中不断优化

**关键**：完美是迭代出来的，不是等出来的。

## 日常执行策略

### 早晨：聚焦设定

**关键问题**：今天哪件事最重要？

**操作**：
1. 今天必须完成的1件事是什么？
2. 其他事情都是次要的
3. 优先完成这1件事
4. 完成后再做其他

### 白天：专注执行

**关键原则**：一次只做一件事

**操作**：
1. 关闭所有干扰
2. 专注做一件事
3. 做完再做下一件
4. 不要多任务

### 晚上：复盘反思

**关键问题**：今天做得怎么样？

**操作**：
1. 今天的计划完成了吗？
2. 哪些做得好？
3. 哪些需要改进？
4. 明天如何改进？

## 常见的内耗场景与应对

### 场景一：不知道做什么

**内耗表现**：
- 觉得很多事情都可以做
- 不知道选哪个
- 一直纠结不开始

**应对方法**：
1. 列出所有选项
2. 给每个选项打分
3. 选择得分最高的
4. 给自己设定时间限制，时间到了必须选

### 场景二：害怕做不好

**内耗表现**：
- 觉得自己准备不够
- 害怕失败
- 一直拖延不开始

**应对方法**：
1. 接受第一次不可能完美
2. 把大任务拆解成小任务
3. 从最小的行动开始
4. 在行动中调整

### 场景三：想做太多

**内耗表现**：
- 什么都想做
- 精力分散
- 什么都做不好

**应对方法**：
1. 明确核心目标
2. 列出所有想做的事
3. 按与目标的相关性排序
4. 只做前3件，其他都砍掉

### 场景四：纠结选择

**内耗表现**：
- 在多个选项中摇摆
- 一直在比较
- 迟迟做不了决定

**应对方法**：
1. 设定决策时间限制
2. 收集必要信息
3. 时间到了必须做决定
4. 接受任何选择都有利弊

## 聚焦的检验标准

### 如何判断自己是否聚焦

**检验问题**：
1. 我能清晰说出自己的核心目标吗？
2. 我每天的时间花在最重要的事情上了吗？
3. 我是不是同时在做太多事情？
4. 我是不是经常感到焦虑和疲惫？

**如果答案是否定的，说明你需要更聚焦。**

### 聚焦的三个层次

**层次一：目标聚焦**
- 有明确的1-3个核心目标
- 其他目标都暂时放下
- 所有行动围绕核心目标

**层次二：时间聚焦**
- 每天有明确的1件最重要的事
- 优先完成这1件事
- 保护大块时间不被打扰

**层次三：精力聚焦**
- 一次只做一件事
- 深度工作，不是浅层忙碌
- 保持精力在重要事情上

## 洋哥的行动建议

### 建议一：用行动消除焦虑

**核心**：想得再多，不如行动一次。

行动能告诉你：
- 你的假设对不对
- 你的方法行不行
- 你的方向准不准

行动是消除焦虑和不确定性的最好方法。

### 建议二：快速试错，快速迭代

**核心**：不要追求一次就做对。

1. 先快速做一个版本
2. 投入市场验证
3. 收集真实反馈
4. 快速迭代优化

失败是成功的一部分，快速试错才能快速成功。

### 建议三：相信长期主义

**核心**：持续做正确的事，时间会给你答案。

- 不要追求速成
- 不要追求一夜暴富
- 相信复利的力量
- 持续做正确的事

长期主义不是慢慢来，而是持续地做正确的事。
