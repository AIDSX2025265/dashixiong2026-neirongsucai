# 推特素材仓库模板

仓库：`https://github.com/AIDSX2025265/dashixiong2026-neirongsucai.git`

## 目录规则

- 每天一个文件夹
- 文件夹名格式：`YYYY-MM-DD`
- 例如：`2026-03-25/`

## 每日目录建议结构

```text
2026-03-25/
  01-素材总表.md
  02-选题清单.md
  03-单条素材-xxx.md
  04-单条素材-xxx.md
```

## 链接规则（重要）

### 放原链接
仅限以下情况：
- 原帖里有 skill 使用链接
- 原帖里有工具地址 / 仓库地址
- 原帖里有命令 / 配置 / 教程入口
- 原帖里有具体步骤、可执行操作、排查细节

### 不放原链接
以下情况不放：
- 纯观点
- 纯判断
- 纯情绪表达
- 纯趋势感受
- 只有启发，没有操作细节

## 默认写作原则
- 不要把干货帖写成纯观点
- 有操作细节就保留操作细节
- 有落地链接才保留原链接
- 优先沉淀成可复用素材，而不是聊天记录
